## 能玩什麼

- **屬性相剋** — 一個招式打不同屬性，是幾倍傷害？
- **種族值 · 誰比較快** — 兩隻二選一比速度，還能選難度；內附「速度線表」按鈕，依賽季查每個種族值在 Lv50 的最速／準速／圍巾／順風實數值。
- **我是誰** — 看黑影猜寶可夢，把中文名字一字不漏打出來（名字裡的英數容許全形與大小寫差異，例如 3d龍＝3D龍）。可挑**世代/地區**（每代涵蓋完整國家圖鑑，含未進化）、**洗翠地區**，或冠軍最新賽季；難度分超級簡單（立繪直接亮著、提示同簡單）、簡單（黑影、露第一個字、其餘圈圈）、普通（黑影、只給圈圈看字數）、困難（含 Mega、黑影、無提示）。計分方式多一個「**按字計分**」（對幾個字給幾分、順序要對；其他測驗只有不計分／正常計分）。另附「圖鑑」可依世代/地區或賽制瀏覽題庫縮圖、用屬性過濾、點立繪看名字。
- **揪朋友比拚** — 把成績碼或連結丟給朋友，玩同一份題目較高下。每種測驗都可選 **10–20 題** 與**計分方式**：不計分（只顯示答對幾題）／正常計分（換算 100 分）；我是誰另可選「按字計分」。
