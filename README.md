## 能玩什麼

開場分兩個入口：**寶可夢快問快答**（測驗，可分享成績碼）與 **寶冠軍工具箱**（對戰查表小工具）。

### 寶可夢快問快答

- **屬性相剋** — 一個招式打某屬性，是幾倍傷害？
- **種族值 · 誰比較快** — 兩隻比速度二選一，可選難度（含隨機）。
- **我是誰** — 看黑影猜寶可夢、把中文名打出來；可挑世代／地區／賽季與難度，另有「按字計分」、自選清單的**出題模式**，與走完整個範圍的**寶可夢大師模式**。
- **寶可夢數獨** — 九宮格各填一隻同時符合直行與橫列條件的寶可夢；含自己挖坑考朋友的**挖坑模式**。玩法參考 [PokeDoku](https://pokedoku.com/)。
- **揪朋友比拚** — 每種測驗都能選 **10–20 題**與計分方式（不計分／正常計分，我是誰另有按字計分），把成績碼或連結丟給朋友挑同一份。

### 寶冠軍工具箱

- **相剋表** — 查任意攻防屬性的傷害倍率，附完整相剋表。
- **聯防小工具** — 把屬性或一支隊伍當整體，看攻守覆蓋與漏洞，並從 meta 給補洞建議。
- **速度線表** — 依賽季查各速度種族值在 Lv50 的最速／準速／圍巾／順風實數值。
- **圖鑑** — 依世代／賽制瀏覽縮圖，可用屬性過濾或**打名字模糊搜尋**，點立繪看名字。
