# 寶可夢快問快答（pokemon-zarbi）

純前端的寶可夢知識快問快答，手機友善。做完一份 10 題測驗會拿到一組**成績碼**，把帶碼的網址（或純代碼）丟給朋友，就能玩**同一份題目**比拚高下。

> **線上試玩**：<https://ahchao.github.io/pokemon-zarbi/>

## 怎麼玩

選一種測驗、答完 10 題，看分數與逐題詳解。想找人較量就把結果頁的成績碼分享出去。

## 測驗類型

1. **屬性相剋** — 一個招式打不同屬性時的傷害倍率（含約三成雙屬性題，會考到 ¼× / 4×）。
2. **種族值 · 誰比較快** — 兩隻寶可夢二選一，純比種族值速度（不計天氣、招式、特性）。可選**難度**：簡單（速度差 20 以上，一眼看出）、中等（差 6～19）、困難（差 5 以內，考驗熟練度）。題庫依《寶可夢 冠軍》賽制名單，**Mega 視為獨立個體**。

另有**屬性相剋查詢**：選攻擊／防禦屬性（可雙屬性）即時看倍率，附完整 18×18 相剋表。

## 成績碼 / 一起測

分享的網址帶 `?c=<碼>`，碼裡編入了測驗類型、賽季、難度、題目 seed、題數與分數。相同 seed 會用確定性亂數重現**完全相同的題目**：

- **挑戰朋友**：分享你做完的成績碼，對方會看到你的分數、玩同一份題目，結果頁直接比輸贏。
- **一起測**：開局前的準備畫面就能拿到「題目碼」（分數記 0），兩人各自貼上同一組碼，拿到同一份題目同步開答。
