## 能玩什麼

- **屬性相剋** — 一個招式打不同屬性，是幾倍傷害？
- **種族值 · 誰比較快** — 兩隻二選一比速度，還能選難度（含「隨機」：每題隨機混合簡單／中等／困難的速度差距）；內附「速度線表」按鈕，依賽季查每個種族值在 Lv50 的最速／準速／圍巾／順風實數值。
- **我是誰** — 看黑影猜寶可夢（電視動畫風格的球框），把中文名字一字不漏打出來（名字裡的英數容許全形與大小寫差異，例如 3d龍＝3D龍；多邊獸Ⅱ 也吃「2」）。可挑**世代/地區**（每代涵蓋完整國家圖鑑，含未進化）、**洗翠地區**，或冠軍最新賽季；難度分超級簡單（立繪直接亮著、提示同簡單）、簡單（黑影、露第一個字、其餘圈圈）、普通（黑影、只給圈圈看字數）、困難（含 Mega、黑影、無提示）。提示難度（超級簡單／簡單）下地區形態打不打地區名都算對（阿羅拉九尾→「九尾」亦可），其他難度要完整名。計分方式多一個「**按字計分**」（對幾個字給幾分、順序要對；其他測驗只有不計分／正常計分）。另附「圖鑑」可依世代/地區或賽制瀏覽題庫縮圖、用屬性過濾、點立繪看名字。還能：
  - **出題模式** — 自己從圖鑑挑一份清單，產生代碼傳給朋友來猜。
  - **寶可夢大師模式** — 本機走完整個範圍，每隻在答對前都會再出現，全部答對就恭喜成為寶可夢大師並統計失誤幾次（需用瀏覽器開啟才會存進度）。
- **寶可夢數獨** — 九宮格，三列三行各是一個條件（地區／屬性／純單屬性／Mega／名字字數）。進去前可選「提示模式」（搜尋清單挑）或「無提示模式」（直接打完整名字）。點任一格、填一隻**同時**符合該列與該行的寶可夢就算對（雙屬性只要中一個就行；同一隻不能重複用）。九格各一次機會，看能對幾題；可「公布解答」在每格旁顯示參考答案（不蓋掉你填的），也能把成績碼丟給朋友挑同一盤。另有「**挖坑模式**」：你先把九格填好當「坑」，產生代碼傳給朋友，朋友要避開你挖的坑、每格選出別的合法答案。玩法參考 [PokeDoku](https://pokedoku.com/)，為中文玩家改作。
- **揪朋友比拚** — 把成績碼或連結丟給朋友，玩同一份題目較高下。每種測驗都可選 **10–20 題** 與**計分方式**：不計分（只顯示答對幾題）／正常計分（換算 100 分）；我是誰另可選「按字計分」。
